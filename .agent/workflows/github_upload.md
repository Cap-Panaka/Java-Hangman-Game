Follow these steps to upload your project to GitHub using professional Git practices.

### 1. Initialize Git and add a `.gitignore`
// turbo
1. Open your terminal in the project directory (`d:\Bro\61.HangManGame`).
2. Run `git init`.
3. Ensure you have a `.gitignore` file to keep your repo clean (I've already created one for you).

### 2. Stage and Commit your files
Use a meaningful commit message.
// turbo
4. Run `git add .` to stage all files.
5. Run `git commit -m "feat: initial commit with Pokemon theme and professional README"`

### 3. Connect to GitHub and Push
6. Run `git branch -M main` (to ensure the branch name is `main`).
7. Run `git remote add origin https://github.com/Cap-Panaka/Java-Hangman-Game.git`
8. Run `git push -u origin main`

### 4. Verify on GitHub
Visit your repository at [GitHub](https://github.com/Cap-Panaka/-Java-Hangman-Game) to see your code, the professional README, and the clean file structure.
